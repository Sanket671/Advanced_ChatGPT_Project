🚀 How to Run Everything
Step 1: Setup (One-Time)

cd python-service

# Create virtual environment
python -m venv venv

# Activate (Windows: venv\Scripts\activate)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Verify
python -c "import transformers; print('✅ Ready')"

Step 2: Run All Tests ⭐

# Install pytest (if not already done)
pip install pytest pytest-cov

# Run full test suite
pytest test_app.py -v

# With coverage report
pytest test_app.py -v --cov=app --cov-report=html

# View coverage
open htmlcov/index.html  # Mac
xdg-open htmlcov/index.html  # Linux
start htmlcov/index.html  # Windows

Expected output:

test_app.py::TestSentimentAnalysis::test_sentiment_positive PASSED
test_app.py::TestSentimentAnalysis::test_sentiment_negative PASSED
...
========================== 50+ passed in 28.34s ==========================

Step 3: Run the Server

# Start Flask server
python app.py

# Expected output:
# INFO ... AI PREPROCESSING ENGINE - Starting Flask server
# INFO ... 🚀 Server running on http://0.0.0.0:5000
# INFO ... 📝 API Endpoint: POST /process
# INFO ... 💓 Health Check: GET /health

Step 4: Test the API (in another terminal)

# Health check
curl http://localhost:5000/health

# Process greeting
curl -X POST http://localhost:5000/process \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello! How are you?"}'

# Process technical issue
curl -X POST http://localhost:5000/process \
  -H "Content-Type: application/json" \
  -d '{"message": "Database connection error"}'

Expected response:

{
